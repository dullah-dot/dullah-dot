### About Me

Saya membangun solusi digital lewat kode — mulai dari web development, otomasi IoT, hingga pengelolaan infrastruktur server.

- **Web Development** — memodernisasi antarmuka lama menjadi tampilan yang bersih dan mudah dipakai, mulai dari landing page statis hingga sistem dengan backend dan panel admin.
- **IoT & Automation** — merancang sistem yang menjembatani hardware dan software.
- **Cloud & Infrastructure** — berpengalaman deploy di AWS dan pengelolaan server (pernah menjabat sebagai admin server sekolah).

**Proyek yang pernah dikerjakan:**
- Merevitalisasi website sekolah yang usang menjadi antarmuka modern dan minimalis.
- Membantu setup teknis untuk kebutuhan sekolah: instalasi CCTV, konfigurasi access point, dan infrastruktur jaringan lainnya.

**Pencapaian akademik:** juara kompetisi Biologi, Bahasa Indonesia, dan Ilmu Kesehatan — memadukan logika dengan kreativitas di luar bidang teknis.

Terbuka untuk kolaborasi proyek open-source, inovasi IoT, atau apa pun yang menantang untuk dikerjakan.
