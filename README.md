### About Me

I build digital solutions through code — spanning web development, IoT automation, and server infrastructure management. I focus on turning outdated or purely functional systems into interfaces that are clean, fast, and genuinely usable for the people who rely on them day to day.

- **Web Development** — Designing and building admin dashboards, client landing pages, and full-stack systems (frontend + backend + database).
- **IoT & Automation** — Bridging hardware and software using microcontrollers like ESP32 and Arduino.
- **Cloud & Infrastructure** — Hands-on experience with AWS deployment, network configuration, and server administration.

Open to open-source collaboration, IoT innovation, or anything that's worth the challenge.

### Featured Projects

**KKB BRI — Vehicle Financing Landing Page & Admin Dashboard**
Designed the frontend architecture and UX flow for a vehicle-financing product landing page paired with an admin dashboard. Focused on understanding the target customer profile to shape an information flow that was easy to read and simple to navigate.

**Alumni Tracking System — SMKN 1 Seyegan (TKJ Department)**
Built as full-stack developer (frontend + backend) to help vocational teachers track alumni employment/education status without manually contacting each student. Includes a student dashboard with a built-in chat feature enabling alumni-to-alumni communication.

**Graduation Eligibility System — TKJ Department**
Collaborated with department instructors to build a system for determining student graduation eligibility, tailored specifically to TKJ (Computer & Network Engineering) program requirements.

**Report Card (Rapor) System**
Worked as full-stack developer designing the data flow and redesigning the report card layout for clearer readability — improving the experience for both students and school administrators.

**Katering Nusantara — Catering Management Platform**
An independent full-stack boilerplate product (Next.js + Express.js + MySQL) featuring role-based dashboards for kasir (cashier), dapur (kitchen), and logistik (logistics), including BOM aggregation, order tracking, and real-time notifications.
